Системные требования

Модификация работает поверх оригинальной игры. Если на вашем компьютере уже запускается «Бесконечное лето», Kanistra MainWorks запустится без дополнительных требований.

Параметр Значение
Базовая игра Бесконечное лето (Everlasting Summer)
Движок Ren'Py
ОС Windows / Linux / macOS
Место на диске Уточняется ближе к релизу

Как установить

  1. Скачайте архив с модификацией по ссылке выше.
  2. Откройте папку с установленной игрой «Бесконечное лето».
  3. Перейдите в подпапку game/mods (если её нет — создайте).
  4. Распакуйте содержимое архива в эту папку.
  5. Запустите игру — модификация появится в меню выбора модов.

Пользователи Steam-версии могут просто подписаться на модификацию в Workshop — установка произойдёт автоматически при запуске Steam.
